The Application

  • The Shiny application developed for the Capstone assignment predicts the next word in a series of word provided by the user. A model was trained using a corpus of news, blogs and twitter data.
  • The Slidify presentation pitch can be found here
  • The Shiny application can be found here

Application Structure, Source code and Model

  • The calculation done on the server are located in the file server.R locate in github here
  • The application build one linear regression model per cut quality
  • Each model is then represented with its error on the scatter plot
  • Models can be isolated by selected the type of cut in the legend
  • THe model will be statistical based on a n-grams approach see this article
